# RFID Based Password Manager (Flask)

A Flask-based password manager secured using RFID authentication.
The web interface unlocks only after scanning a valid RFID card via serial communication.

## Features
- RFID-based access control
- Password generation & management
- Flask web UI
- Google Drive cloud sync
- Clean, table-based UI

## Tech Stack
- Python
- Flask
- ESP + RFID
- Google Drive API

## Note
Sensitive files (tokens, passwords) are ignored using .gitignore.
